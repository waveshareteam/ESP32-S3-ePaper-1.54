由于该款墨水屏只能显示黑白，所以我在字体和图形上处理了些东西，

1.最直接的方式就是，使用我们提供的xiaozhi固件
2.如果需要增加功能，编译xiaozhi的时候，需要替换78__xiaozhi-fonts目录，不然图形显示可能会异常


Since this e-paper display can only show black and white, I made some adjustments to the fonts and graphics.

The most straightforward way is to use the xiaozhi firmware we provide.

If you need to add more features, when compiling xiaozhi, you must replace the 78__xiaozhi-fonts directory, otherwise the graphics may not display correctly.
